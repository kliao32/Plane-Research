# Aircraft Survival Analysis

## Overview
This project analyzes how long aircraft remain active using survival analysis.

## Data
Data is not included due to size. It can be obtained from public aircraft datasets.

## Methods
- Kaplan-Meier survival curves
- Log-rank test
- Logistic regression

## Results
- Aircraft are less likely to remain active as they age  
- Significant differences exist between aircraft models  

## Files
- Plane.ipynb: main analysis notebook  
- Plane Research Report.pdf: full report
